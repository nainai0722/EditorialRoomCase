# 離婚記事プロジェクト

noteで離婚までの経緯を発信するプロジェクトの管理リポジトリ

## フォルダ構成

- **drafts/** - 執筆中の記事下書き
- **published/** - 公開済み記事のバックアップ
- **ideas/** - 記事アイデア・ネタ帳
- **analytics/** - 反響データ・分析
- **templates/** - 記事テンプレート

## 使い方

1. `ideas/ideas.md` で次に書きたいテーマを整理
2. `templates/article-template.md` をコピーして新しい記事を下書き
3. `drafts/` で執筆・添削
4. 公開後、`published/` にバックアップ
5. `analytics/reactions.md` に反響を記録
