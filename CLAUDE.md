# 離婚記事プロジェクト - 執筆ガイドライン

## プロジェクト概要

## 最重要指示（Claude向け）

- 本ドキュメントは「厳守ルール」と「素材」を含む
- 以下は必ず守ること：
  - 文体・トーンガイドライン
  - 避けるべき表現
  - 記事構成のルール
- 登場人物設定・エピソード集は「素材」であり、
  すべてを使う必要はない

### 目的
10年間の夫婦生活から別居・離婚に至るまでの経緯を、
感情に溺れず構造的に描写し、結果としてドラマ性を生み、
noteでマネタイズする。

### マネタイズ戦略
- 1-10記事：完全無料（読者獲得）
- 11記事目以降：導入部のみ無料、続きは有料
※ 有料パートでは感情描写よりも、
具体的な出来事・やり取り・判断を重視する。

### ターゲット読者
- ママ層（特に離婚を考えている人、夫婦関係に悩む人）
- 体験談を読みたい人
- 既婚者・独身問わず共感できる内容

### 投稿ペース
週末ごと、または2日おき

### 上野駅のエピソード（離婚の決め手）

**状況:**
- 雨の上野駅
- 子どもをパンダ（上野動物園）に連れて行こうとしていた

**わたしの思考:**
- 子どもにパンダを見せたい
- でもそれ以上に「夫の機嫌を損なわないように」に必死
- 夫の疲労具合を見て、機嫌良くなってもらう最適解を探している

**決定的な気づき:**
- 子どもが「おしっこ」と言った
- わたしは即座に夫の様子を見てしまう
- トイレなのに、子どもより夫の機嫌を優先している異常さ
- この異常さを言語化してしまった瞬間、離婚を決意

**キーフレーズ:**
「子どものトイレすら、夫の機嫌次第」

## 文体・トーンガイドライン

### 基本方針
- **素直 + 皮肉**
- 感情的になりすぎず、冷静に事実を述べる
- 時々ユーモアを挟む
- 具体的な数字を多用する

### 特徴的な表現

**心の中のおじさん:**
- わたしの苦悩を俯瞰的に伝えてくれる存在
- 「この若造が金の無心をしてきたのは、1年半ぶりだったかな」
- あごひげを指でなで上げる仕草

**比喩:**
- 「メンブレホス落ち女子がホストに散財するように」
- 「供物に差し出すような心地」
- 「令和羅生門の追い剥ぎババア」
- 「令和の呪わしい知性的なツール」（ChatGPT）

**自虐的ユーモア:**
- 「ゴミのような口上」
- 「謎のプロテイン」
- 「ワイヤレスイヤホン難民」
- 「不思議だよね？」（失業手当の話）

**数字を使った説得力:**
- 7年間で180万円
- 月9万の浪費
- 化粧品3つだけ
- 体重47キロ

### 避けるべき表現

❌ 過度に感情的な表現（「許せない！」「最低！」など）
❌ 夫を一方的に悪者にする表現
❌ 自分を完全な被害者として描く
❌ 読者に同情を求める表現
❌ 絵文字（使わない）

### 推奨する表現

✅ 事実を淡々と述べる
✅ 自分の気持ちを冷静に分析する
✅ ユーモアを交える
✅ 具体的な数字で説得力を出す
✅ 「でも」「だが」「しかし」で逆接を使う

## 記事構成のルール

### 基本構成

1. **導入（インパクト重視）**
   - 衝撃的な事実から始める
   - または印象的なエピソードから始める
   - 読者を引き込む

2. **背景説明**
   - 家族構成、年収などの基本情報
   - 簡潔に

3. **エピソード展開**
   - 具体的なエピソードを2-3個
   - 短い段落で読みやすく
   - 太字で重要なポイントを強調

4. **気づき・転機**
   - 「そのとき気づいた」「言語化してしまった」
   - エピソードから得た教訓

5. **現在・決意**
   - 別居・離婚に至った経緯
   - 前向きな締め

6. **次回予告**
   - 続きが気になる仕掛け

### 段落の長さ
- 1段落は2-5行程度
- スマホで読みやすい短さ

### 見出し（##）の使い方
- noteでは##が太字になる
- 重要なセクションの区切りに使う
- 5-10文字程度

### 太字（**）の使い方
- 重要なキーフレーズ
- 数字
- 衝撃的な事実

### 次回予告の例
```
**次回予告:**
夫は休日、起きない話を書きます。
```

## ChatGPT/AI の使い方（メタ的な記述）

### プロジェクト内での役割
- 記事の下書き作成
- 添削・ブラッシュアップ
- 構成アドバイス
- 次の記事の提案

### AIとの協働姿勢
- AIの回答を鵜呑みにしない
- 自分の言葉で書く
- AIは叩き台として使う
- 最終的な判断は自分でする

## 注意事項

### プライバシー保護
- 実名は使わない
- 具体的な地名は避ける（「東京・上野駅」程度はOK）
- 夫の職業の詳細は書かない

### 法的リスク回避
- 名誉毀損にならないよう、事実のみを書く
- 過度に夫を貶めない
- 暴力の描写は事実を淡々と述べる

### 子どもへの配慮
- 子どもが読んでも傷つかない表現
- 子どもを過度に巻き込まない
- 「子どもかわいそう」と思わせない
